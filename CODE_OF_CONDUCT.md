# 行動規範

Acecore の各プロジェクトは、実務的で敬意ある協働の場として運用します。
Issue、Pull Request、レビュー、コメント、その他のプロジェクト上のやり取りに
参加する人は、この行動規範に従ってください。

## 望ましい行動

- 相手への敬意を保ち、建設的にやり取りする。
- 技術的な意見の違いは、作業内容・根拠・トレードオフに絞って扱う。
- 公開コメントは将来の貢献者や利用者にも読まれる前提で書く。
- 変更を提案するときは、レビューしやすいよう背景と目的を共有する。
- プライバシー、機密情報、法的な境界を尊重する。

## 認められない行動

- 嫌がらせ、脅迫、個人攻撃、差別的な表現。
- 明確な許可なく個人情報や機密情報を公開すること。
- スパム、繰り返しの脱線投稿、意図的な妨害。
- プロジェクトの決定、所属、貢献者の意図を偽って伝えること。
- 悪用、攻撃、不安全な活動の調整にプロジェクトの場を使うこと。

## 対応

メンテナーは、プロジェクトを守るために必要な範囲で、コメントの編集・非表示・
削除、Issue や Pull Request のクローズ、参加制限、アカウントのブロックを行う
場合があります。

問題を見つけた場合は、リポジトリで案内されている適切な非公開連絡手段を使って
ください。リポジトリ固有の案内がない場合は、Acecore のWebサイトにある
お問い合わせフォームを利用してください。
